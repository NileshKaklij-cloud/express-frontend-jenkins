# Express Frontend — VSA Infotech

## Description
Student Management web UI built with Express.js + EJS templates.

## Setup
```bash
npm install
```

## Run
```bash
node server.js
```

## Environment Variables
| Variable  | Default               |
|-----------|-----------------------|
| FLASK_URL | http://localhost:5000 |
| PORT      | 3000                  |
